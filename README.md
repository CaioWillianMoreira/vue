## Curso vueJs

- Instancia um componente vue

```
new Vue ({
})
```
- Propriedade __el__ diz onde é o container que a sua aplicação está inserida, parecido com "querySelector"

```
  el: '#pp'
```

- Template que será ultilizado

```
template: `seu HTML`
```
### ex :

```
new Vue ({
  el: '#app',
  template: `<html>Seu HTML</html>`
})
```
